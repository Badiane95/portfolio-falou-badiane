## 📦 Installation & Développement

### Prérequis
- Node.js 22+
- pnpm 10+

### Installation
```bash
# Cloner le dépôt
git clone https://github.com/Badiane95/portfolio-falou-badiane.git
cd portfolio-falou-badiane

# Installer les dépendances
pnpm install

# Lancer le serveur de développement
pnpm dev
```

### Build pour Production
```bash
# Compiler le projet
pnpm build

# Prévisualiser la build
pnpm preview
```

### Tests
```bash
# Exécuter les tests vitest
pnpm test
```

## 🔄 Workflow de Développement

1. **Modifier le code** dans `client/src/` ou `server/`
2. **Tester localement** avec `pnpm dev`
3. **Committer les changements** : `git commit -m "Description"`
4. **Pousser vers GitHub** : `git push origin main`
5. **GitHub Pages** se met à jour automatiquement
